> 百科全书是对抗碎片化的方法之一

# 编剧

## 《把人物写活》

这本书，也能带领我们理解周围人(包括教育孩子)。请用旁观者[无论对任何人]的心态生活,不要把自己缺失的东西灌输给别人，然后自以为是爱。

顺人物的性格，不要受到自恋荷尔蒙影响，不要为了让事情合乎逻辑而强制插手，不要推演任何事而思虑过重[在事情发生之前都不算真实]，旁观然后记录。

减少攻击性，尤其不要想象对方有攻击性。

-  每个人都有自己的意志，并不是非黑即白的性格
- 作者不能移情到任何主角身上，避免给主角铺上一条康庄大道【没有痛苦和挣扎的人生是不值得过的，这句话也适用你的亲人】

**一句话总结**

有人一生劳碌有人事不关己，其实做和不做都影响不大[**不要把结局想象的恐怖,因恐惧而做的事情会带来悲剧。注重过程，而非结局**]，关键在于理解和传递信息。
